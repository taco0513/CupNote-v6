# HomeCafe Mode - 화면 4: 테이스팅

## 화면 개요
추출한 커피의 맛을 평가하고 기록하는 단계

## 화면 구성

### 헤더
```
[←] 테이스팅                     (4/5)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
- 진행률: 80% 완료

## 평가 시스템

### 5점 척도
- ⭐: 실패
- ⭐⭐: 아쉬움
- ⭐⭐⭐: 보통
- ⭐⭐⭐⭐: 만족
- ⭐⭐⭐⭐⭐: 완벽

### 맛 프로필 레이더
```
      산미
       5
      /|\
     / | \
단맛 - - - - 쓴맛
    \  |  /
     \ | /
      \|/
      바디
```

### 추출 진단
- 과소/적정/과다
- 균일도 평가
- 청정도 평가
