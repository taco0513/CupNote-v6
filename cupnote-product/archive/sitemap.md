## 네비게이션 구조

### 하단 네비게이션 (Bottom Navigation)
모바일 앱의 주요 네비게이션으로 항상 표시

```
┌─────────────────────────────────┐
│          콘텐츠 영역              │
├─────────────────────────────────┤
│  🏠    ➕    📚    🏆    👤     │
│  홈   기록  내기록  성취  프로필   │
└─────────────────────────────────┘
```

**특징**:
- 5개 주요 섹션 직접 접근
- 현재 위치 하이라이트
- 중앙 FAB 버튼으로 새 기록

### 상단 앱 헤더 (App Header)
```
┌─────────────────────────────────┐
│ ← CupNote             🔍 🔔 ⚙️  │
└─────────────────────────────────┘
```

**구성 요소**:
- 뒤로가기 버튼 (컨텍스트별)
- 앱 로고/타이틀
- 검색 (전역)
- 알림
- 설정 바로가기

## 페이지 계층 구조

### Level 0: 진입점
- 스플래시 화면
- 온보딩 (첫 사용자)

### Level 1: 주요 섹션
- 홈
- 새 기록
- 내 기록
- 성취
- 프로필

### Level 2: 서브 페이지
- 기록 상세
- 분석 대시보드
- 설정
- 검색 결과

### Level 3: 모달/팝업
- 필터 옵션
- 사진 뷰어
- 공유 옵션
- 확인 다이얼로그

## 사용자 플로우

### 주요 Task Flow

#### 1. 새 커피 기록 작성
```
홈 → [+버튼] → 모드 선택 → 정보 입력(다단계) → 저장 → 결과 확인
```

#### 2. 과거 기록 찾기
```
내 기록 → [검색/필터] → 결과 목록 → 상세 보기
```

#### 3. 성취 확인
```
성취 → 카테고리 선택 → 배지 상세 → 진행률 확인
```

#### 4. 프로필 수정
```
프로필 → 편집 버튼 → 정보 수정 → 저장
```

## 콘텐츠 우선순위

### 홈 화면
1. **필수**: 새 기록 버튼
2. **중요**: 최근 기록, 진행 중인 성취
3. **보조**: 팁, 커뮤니티 하이라이트

### 기록 목록
1. **필수**: 커피 이름, 평점, 날짜
2. **중요**: 로스터리, 썸네일
3. **보조**: 태그, 메모 미리보기

### 기록 상세
1. **필수**: 모든 입력 정보
2. **중요**: 사진, 커뮤니티 매치
3. **보조**: 수정 이력, 공유 옵션
