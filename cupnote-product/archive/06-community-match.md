## 매치 점수 알고리즘

### 계산 요소
```javascript
matchScore = {
  ratingMatch: 40,     // 평점 일치도
  tasteMatch: 30,      // 맛 표현 유사도
  preferenceMatch: 20, // 선호도 패턴
  contextMatch: 10     // 상황 유사도
}
```

### 특수 케이스
- **첫 기록자**: 자동 100% + 보너스 포인트
- **소수 기록** (<5명): 조정된 계산식
- **이상치**: 극단값 제외
